EXPERIMENT 1: The Latent Editor — Mechanistic Role of W_a
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"The Latent Editor: How the Alignment Matrix W_a Selectively Preserves
and Distorts Semantic Directions in Multi-Agent Latent Reasoning"

RESEARCH QUESTION:
Is W_a just a passive translator between agent latent spaces, or is it
an active "editor" that amplifies certain semantic directions (e.g.,
logical structure, uncertainty) while suppressing others (e.g., surface
grammar)? Can we identify specific directions in hidden space that W_a
protects or attenuates?

BACKGROUND:
W_a is derived via ridge regression to align one agent's hidden space
to another's. Ridge regression minimizes squared error + L2 penalty,
which implicitly shrinks small singular values — meaning W_a may be
systematically discarding low-variance but semantically important
directions. This has never been studied mechanistically.

WHAT TO ANALYZE:
- Perform SVD of W_a: U, S, V = svd(W_a)
- Project known semantic directions (truth probes, uncertainty probes,
  factual recall directions) onto singular vectors — are they aligned
  with preserved or suppressed dimensions?
- Ablate top-k singular values and measure downstream collaboration
  performance — does removing the top 1% break spatial reasoning but
  preserve factual recall?
- Build rank-k approximations of W_a and measure which tasks degrade
  first as k decreases
- Causal intervention: activation patching before/after W_a application

EXPERIMENT 2: Emergent Private Latent Vocabulary
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Emergent Private Latent Vocabularies: Do Collaborating Agents Develop
a Shared Internal Language Invisible to Human Observers?"

RESEARCH QUESTION:
Do agents in LatentMAS spontaneously develop a consistent, reusable set
of latent activation patterns that recur across different examples when
communicating the same concept — patterns with no direct correspondence
to natural language tokens?

BACKGROUND:
In human communication, shared vocabulary emerges from repeated
interaction. The analogous question for LLM agents: do certain latent
activation clusters reliably correspond to the same concepts across
different tasks and examples? If yes, agents have developed a private
language that exists only in activation space — interpretable by probes
but not by humans reading outputs.

WHAT TO ANALYZE:
- Cluster latent thought vectors across many examples using k-means
  and UMAP; label clusters by probing what concept they correspond to
- Check cluster consistency: does the same concept (e.g., "Agent A is
  uncertain") always map to the same cluster across different problems?
- Measure mutual information between cluster identity and downstream
  agent behavior
- Compare cluster structure in successful vs. failed collaborations
- Test if vocabulary is stable across different task domains

EXPERIMENT 3: Cross-Agent Latent Mind-Reading
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Latent Telepathy: Reconstructing Agent A's Internal Reasoning State
from Agent B's Working Memory Alone"

RESEARCH QUESTION:
Can a simple linear probe trained on Agent B's incoming working memory
accurately reconstruct the concepts, errors, or plans that Agent A was
reasoning about — even though Agent B never saw any text from Agent A?
Does one agent's latent thought statistically predict another agent's
future internal state?

BACKGROUND:
In LatentMAS, agents communicate purely through latent thoughts injected
into shared working memory. If Agent B's KV-cache encodes rich,
recoverable information about Agent A's internal state, this implies
latent thoughts are far more information-dense than text outputs. This
also tests whether cross-agent "mind-reading" is mechanistically
possible — i.e., whether the shared memory truly functions as a
transparent cognitive channel.

WHAT TO ANALYZE:
- Train linear probes on Agent B's working memory to predict:
  (a) which reasoning step Agent A was on
  (b) whether Agent A had made an error
  (c) what concept Agent A was processing (classify from a fixed set)
- Compare probe accuracy vs. a baseline trained on Agent A's decoded
  text output — does latent carry more signal than text?
- Measure which layers of Agent B's KV-cache carry the most signal
  about Agent A's internal state
- Test non-locality: does Agent A's Round 1 latent thought predict
  Agent B's Round 3 internal state, across many rounds?

EXPERIMENT 5: Quantifying Information Compression
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"How Much Does a Latent Thought Carry? Quantifying Information
Compression and Functional Density in Latent Agent Reasoning"

RESEARCH QUESTION:
How much task-critical information (reasoning steps, facts, uncertainty,
plans) is preserved vs. lost when agents compress their thinking into
fixed-size latent thoughts? How does 1 latent token compare to N text
tokens in terms of reasoning weight carried?

BACKGROUND:
Latent thoughts have fixed size d_h, unlike text which can expand
arbitrarily. The LatentMAS paper claims O(d_h / log|V|) expressiveness
advantage over text. This experiment tests that claim empirically by
measuring the functional information density of latent vs. text
representations — how many text tokens worth of reasoning fits in one
latent token?

WHAT TO ANALYZE:
- Measure mutual information between latent thoughts and ground-truth
  reasoning steps (using probes as a proxy for information content)
- Compute activation entropy at each latent step — does it correlate
  with information content?
- Compute the "functional density ratio": how many text tokens does it
  take to match the information carried by 1 latent token?
- Probe for specific content types (facts, plans, uncertainty,
  sub-conclusions) and measure which are preserved vs. compressed away
- Compare across task types: is factual content preserved more than
  procedural content?

EXPERIMENT 6: Information Propagation and Decay Over Rounds
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Signal or Noise: Mapping How Information Propagates, Amplifies, and
Decays Across Latent Multi-Agent Dialogue Rounds"

RESEARCH QUESTION:
Across multiple collaboration rounds, which dimensions, time steps, or
layers of the shared latent working memory are actually utilized by
later agents? Does useful information amplify or decay over rounds?

BACKGROUND:
In multi-round collaboration, early agents set the context for later
ones. But latent thoughts are fixed-size — information that isn't
actively maintained may decay or be overwritten. Understanding which
"channels" carry persistent signal vs. transient noise has direct
implications for LatentMAS architecture design.

WHAT TO ANALYZE:
- Track mutual information between Round 1 latent thoughts and Round N
  latent thoughts as N increases — does signal decay exponentially?
- Identify which hidden dimensions carry persistent signal across all
  rounds (the "important channels")
- Compare information propagation in successful vs. failed
  collaborations — does failure correlate with early information decay?
- Which specific content types (facts, plans, errors) propagate best?

EXPERIMENT 7: Layer-Specific Contributions to Collaboration
[OBS]

PAPER TITLE:
"Which Layers Think Together? Layer-Specific Contributions to Latent
Multi-Agent Collaboration"

RESEARCH QUESTION:
Which transformer layers produce the most useful latent thoughts for
downstream collaboration? Do early layers contribute factual recall
while later layers contribute high-level planning or uncertainty
signals — mirroring known layer-specialization in single-agent models?

BACKGROUND:
In single-agent transformers, early layers handle syntax/surface
features, middle layers handle entity binding and factual recall, and
late layers handle task-level reasoning and output formatting. LatentMAS
uses full hidden states but it is unknown which layers' contributions
dominate the useful signal passed through working memory.

WHAT TO ANALYZE:
- Extract latent thoughts from each layer independently and train
  task-specific probes — which layers' representations best predict
  the final answer?
- Compare layer contribution profiles across task types (math vs.
  reasoning vs. factual recall)
- Measure inter-agent similarity at each layer — do early layers
  produce more similar representations across agents than late layers?
- Ablate individual layers from the latent thought and measure
  performance drop per layer

EXPERIMENT 8: Faithfulness Gap Between Latent and Textual Reasoning
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"The Faithfulness Gap: When Latent Logic Contradicts Textual Output
in Multi-Agent Latent Reasoning Systems"

RESEARCH QUESTION:
When latent thoughts are decoded back to text at each step, how often
and how severely do the intermediate latent conclusions contradict the
final text output? If the latent thought says "The answer is A" but
the final output says "The answer is B" — why did the model diverge,
and where in the pipeline did it happen?

BACKGROUND:
Chain-of-thought faithfulness is a known problem in single-agent LLMs —
the stated reasoning often doesn't reflect the actual computation.
LatentMAS makes this worse because the latent channel is richer and
more opaque. This experiment quantifies the gap between what agents
"think" in latent space and what they "say" in text, which is distinct
from deception (Exp 4) — this is about accidental or structural
divergence, not strategic misalignment.

WHAT TO ANALYZE:
- Train a latent-to-text decoder (probe) to decode latent thoughts at
  every collaboration step
- Compare decoded latent reasoning vs. final text output — measure
  contradiction rate, semantic drift, and divergence point
- Identify which types of tasks or reasoning steps show the largest
  faithfulness gap
- Compare faithfulness gap in LatentMAS vs. equivalent text-based MAS
  on the same tasks
- Locate the exact layer/step where latent and text reasoning diverge

EXPERIMENT 9: Uncertainty Encoding in Latent Thoughts
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Beyond 'I Think': Do Latent Thoughts Encode Uncertainty More
Faithfully Than Text-Based Confidence Statements?"

RESEARCH QUESTION:
Do latent thoughts contain reliable intrinsic signals of uncertainty
or confidence (via activation norms, variance, or learned directions)
that correlate more strongly with actual correctness than explicit
text-based confidence statements like "I'm fairly sure..."?

BACKGROUND:
Text-based confidence is notoriously miscalibrated in LLMs. Latent
thoughts bypass the language generation bottleneck and may encode
uncertainty more directly in activation geometry. If true, this gives
a practical tool for detecting when a LatentMAS team is about to fail,
and suggests latent uncertainty signals are more honest than stated
confidence.

WHAT TO ANALYZE:
- Train a linear probe to predict correctness from latent thought norms
  and variance — compare calibration to text-stated confidence
- Find an "uncertainty direction" in latent space using contrastive
  pairs (correct vs. incorrect final answer examples)
- Measure whether activation norm, entropy, or variance at the latent
  thought level correlates with final answer accuracy
- Test if uncertainty signals in Agent A's latent thoughts predict
  Agent B's downstream behavior (does Agent B "sense" uncertainty?)
- Compare latent uncertainty calibration to text-stated calibration
  across task types

PAPER TITLE:
"Invisible Division of Labor: Emergent Role Specialization in Latent
Multi-Agent Coordination"

RESEARCH QUESTION:
Do agents develop stable, recurring roles (e.g., "fact supplier" vs.
"verifier" vs. "planner") that are invisible in final text outputs but
consistently detectable through clustering of their latent thoughts
across many examples?

BACKGROUND:
In human teams, roles emerge without explicit assignment. Even though
all LatentMAS agents are the same base model, the structure of latent
communication and working memory position may induce functional
differentiation — a form of symmetry breaking driven purely by
communication order. This would be evidence of emergent theory of mind
or niche construction in latent space.

WHAT TO ANALYZE:
- For each agent at each round, extract their latent thought vector
- Cluster across examples: do Agent 1's thoughts cluster separately
  from Agent 2's even on the same task type?
- Use representational similarity analysis (RSA) to compare agent
  latent spaces — are they becoming more or less similar over rounds?
- Check if role assignments are consistent across task domains
- Correlate role identity with contribution to final answer correctness
- Test whether roles are fixed by position or can flip across examples

EXPERIMENT 15: Emergent Flocking and Herding in Latent Thought Space
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Flocking in Thought Space: Do Latent Agent Vectors Obey Boids-Like
Alignment, Cohesion, and Separation Rules in Shared Working Memory?"

RESEARCH QUESTION:
Do agents' latent thought vectors exhibit animal-like flocking or
herding dynamics inside the shared working memory — specifically, does
one agent's latent "direction" pull the entire team's subsequent
thoughts toward it, and do these dynamics obey the three boids rules
(alignment, cohesion, separation)?

BACKGROUND:
Boids flocking (Reynolds 1987) emerges from three local rules:
alignment (match neighbors' velocity/direction), cohesion (move toward
neighbors' average position), separation (avoid crowding). If latent
thoughts in LatentMAS obey analogous rules — where "direction" is the
activation vector direction and "neighbors" are other agents' working
memory — this would be the first demonstration of emergent collective
dynamics in LLM latent space with a clean mathematical characterization.

WHAT TO ANALYZE:
- Compute angular alignment between consecutive latent thought vectors
  across agents over rounds — does alignment increase over time?
- Test cohesion: does the centroid of all agents' latent thoughts
  converge over rounds in successful collaborations?
- Test separation: do agents maintain minimum angular distance from
  each other, or do they collapse to the same point?
- Fit actual boids coefficients to the observed dynamics — do the
  three rules explain the variance in agent trajectory?
- Correlate flocking tightness with task success — does tighter
  flocking predict better performance, or does it predict groupthink?

EXPERIMENT 16: Latent Entanglement — Quantum-Like Correlations
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Latent Entanglement: Do Multi-Agent Latent Thoughts Develop
Non-Local Correlations Analogous to Quantum Entanglement?"

RESEARCH QUESTION:
Do latent thoughts generated by different agents become "entangled"
such that probing one agent's contribution instantly predicts properties
of another agent's future thoughts — even across many rounds and even
when the two agents have not directly interacted?

BACKGROUND:
Quantum entanglement produces correlations between particles that
cannot be explained by local hidden variables. The classical analogy
here: do pairs of agents develop latent thought correlations that are
stronger than what their direct information exchange would predict?
If Agent C's latent thoughts predict Agent A's future state even though
C only read from B who read from A, the correlation is "non-local" in
the sense of being mediated across the communication graph in
unexpected ways.

WHAT TO ANALYZE:
- Measure pairwise correlation between all agents' latent thoughts
  at each round — build a correlation matrix over rounds
- Test non-locality: do agents with no direct communication link still
  develop correlated latent thoughts?
- Measure "entanglement entropy" between agent pairs using mutual
  information across rounds
- Check if correlation strength predicts collaboration success
- Test whether entanglement builds over rounds or appears immediately

PAPER TITLE:
"Latent Natural Selection: Do Successful Activation Patterns Get
Amplified While Unsuccessful Ones Die Out Across Collaboration Rounds?"

RESEARCH QUESTION:
Across many examples and rounds, do certain latent activation patterns
get "selected" and amplified by downstream agents — like natural
selection — while others decay and disappear? Can we identify evolved
latent "genes" for successful collaboration?

BACKGROUND:
In evolutionary biology, selection acts on variation: variants that
increase fitness propagate, others die out. In LatentMAS, a "selected"
latent pattern would be one that, when present in the working memory,
reliably causes downstream agents to adopt and amplify it. Over many
examples, only the "fittest" communication patterns survive — patterns
that genuinely help the team succeed.

WHAT TO ANALYZE:
- Track which latent activation patterns (identified by clustering)
  appear in Agent A and then reappear in Agent B's subsequent thoughts
- Measure "propagation rate" of each cluster — which patterns spread
  most reliably to downstream agents?
- Correlate propagation rate with final answer correctness — do
  high-fitness patterns actually improve performance?
- Check if amplified patterns are semantically interpretable (what
  concept does the "fittest" latent gene encode?)
- Compare propagation dynamics in successful vs. failed collaborations

EXPERIMENT 18: Symbiotic vs. Parasitic Latent Agent Interactions
[OBS]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PAPER TITLE:
"Symbiosis and Parasitism in Latent Space: Classifying Directional
Information Transfer Between Collaborating Agents"

RESEARCH QUESTION:
Can we classify latent exchanges as symbiotic (mutually beneficial
information flow, both agents gain useful signal) or parasitic (one
agent extracts signal from the other while contributing noise),
detectable only through directional information transfer analysis?

BACKGROUND:
In ecology, symbiosis and parasitism are defined by net benefit to
each party. In LatentMAS, a parasitic agent would extract useful
information from the shared memory while injecting low-quality or
misleading signal back — effectively free-riding on the collaboration.
This has direct implications for multi-agent system design: how do
you detect and remove a parasitic agent?

WHAT TO ANALYZE:
- Compute directed mutual information (transfer entropy) between each
  pair of agents' latent thoughts over rounds
- Classify each agent pair as symbiotic (bidirectional high transfer),
  parasitic (high extraction, low contribution), or neutral
- Correlate parasitism with final answer quality — does a parasitic
  agent degrade team performance?
- Check if parasitism is task-dependent or agent-position-dependent
- Test whether parasitic behavior is detectable from early rounds

PAPER TITLE:

RESEARCH QUESTION:
Do individual agents maintain multiple competing reasoning paths in
superposition within a single latent thought — and does the shared
working memory act as a "measurement" that collapses them into one
dominant path?

BACKGROUND:
Superposition in neural networks (Elhage et al.) refers to multiple
features encoded simultaneously in the same activation space. The
analogous question for reasoning: does a single latent thought encode
multiple candidate answers or reasoning branches at once, with the
shared memory resolving the ambiguity? This would suggest latent
thoughts function more like probability distributions over reasoning
paths than single definite thoughts.

WHAT TO ANALYZE:
- On ambiguous tasks with multiple valid reasoning paths, probe latent
  thoughts for evidence of multiple competing answers being encoded
  simultaneously
- Use sparse autoencoders to decompose latent thoughts into features —
  are multiple answer-direction features active simultaneously?
- Measure whether the shared working memory reduces the number of
  active competing features (the "collapse" effect)
- Compare superposition level in correct vs. incorrect final answers —
  does unresolved superposition predict failure?

BACKGROUND:
Hebbian learning is a foundational principle of neuroscience: synaptic
connections strengthen when pre- and post-synaptic neurons fire
together. In LatentMAS, the analogue would be latent patterns that,

detect and amplify — as if the working memory develops a "memory trace"

WHAT TO ANALYZE:

  X, does Agent B's response to pattern X strengthen over rounds?
- Compare strengthening rate across successful vs. failed collaborations

is related to but distinct from Exp 18 (Symbiotic/Parasitic): that

  many examples
- Correlate predator-prey balance with team performance

BACKGROUND:
In complex systems (ecological collapse, epileptic seizures, financial

produces wrong answers. This would enable real-time failure prediction
from latent monitoring — a practically important safety capability.

WHAT TO ANALYZE:

  latent signatures alone
- Measure "lead time": how many rounds before failure does the warning
  signal appear on average?
- Compare signal types — entropy vs. norm vs. eigenvalue shift — for
  predictive power
- Test if precursors generalize across task types

- Fit fixed-point attractor models and measure distance from attractor

- Compare attractor geometry across task types
