OVERVIEW
--------
LatentMAS is a multi-agent system where agents (planner, critic, refiner, judger)
communicate via latent hidden states instead of text. A learned alignment matrix
W_a transforms hidden states between agents. This project interprets what is
actually happening inside that latent communication channel.

Model: Qwen/Qwen3-4B
Tasks: GSM8K (math reasoning), ARC-Challenge (science MCQ), MBPP+ (code generation)
Data collection: final_run.py (13 conditions), final_run_q2.py (confidence-gated)

   ARC: 18.9% (CI: 5.4-35.1%)
   MBPP+: 24.8% (CI: 13.9-35.6%)

I4 Confirmation bias test:
   Cosine shift toward correctness centroid is nearly identical for correct

INTERPRETATION: When failures happen, they tend to occur late in the process
(round 2-3), not at the beginning. Once an error occurs, roughly 18-25% of
subsequent agents are "infected" (diverge beyond threshold). There is no
systematic confirmation bias — agents don't systematically pull toward
the wrong answer more than the right one.

KEY TAKEAWAYS
============================================================

1. W_a IS LIKELY TRIVIAL: Identity W_a ≈ trained W_a on all tasks (no
   statistically significant accuracy difference). W_a may just be
   passing hidden states through with minimal transformation.

2. COLLABORATION WINS ARE RARE ON MATH/SCIENCE: Bucket-1 is only 1-3%
   for GSM8K/ARC (below the ROADMAP's own 5% threshold). LatentMAS does
   not meaningfully outperform single-agent on these tasks.

3. CODE GENERATION IS DIFFERENT: MBPP+ has B1=13% and LatentMAS does
   outperform single-agent (+6.4pp). But TextMAS beats LatentMAS on MBPP+
   (89% vs 73.3%), so text communication is better than latent for code.

4. ROLES ARE DISTINCT BUT REDUNDANT: Agents are 99% identifiable by role
   (E2) but share ~75-80% of their information (K1). Each agent is unique
   but mostly saying the same thing as the others.

5. SYCOPHANCY IS REAL: Failures are characterized by convergence to the
   Critic's (agent 1) representation in 73-88% of cases. Social influence
   dynamics exist in the latent space.

6. LOGIT-LENS DATA IS MISSING: Exp H (faithfulness gap) cannot be computed
   because all logitlens.json files are empty (bug in data collection).

7. HEADLINE CHECK FAILED: best_of_n and cot_matched conditions were not
   run. Cannot report whether LatentMAS beats compute-matched baselines.

8. PROBE CANNOT GATE: The Exp P probe falls back on 100% of examples at
   any useful threshold, making Exp Q2 (confidence-gated) non-functional.
